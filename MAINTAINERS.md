# Maintainers

OSMM is **maintainer-led** (see [`GOVERNANCE.md`](GOVERNANCE.md)): the community
proposes, and a small maintainer core decides. This file lists that core.

[`NOTICE`](NOTICE), [`LICENSING.md`](LICENSING.md), and [`TRADEMARK.md`](TRADEMARK.md)).

The core will grow as additional maintainers are named. Maintainers approve and
merge changes, assign object lifecycle states, cut releases, and own the
controlled vocabularies and the object registry.
